# Markdown Renderer

Tests for the markdown renderer (`--format markdown`).
Output is clean, standard CommonMark-compatible markdown.
MDC/Comark syntax is converted to HTML tags, frontmatter is removed,
and raw HTML blocks are stripped.

## Headings

```````````````````````````````` example [no-normalize]

.
# Heading 1
````````````````````````````````
